Given a list of numbers and a number k,
return whether any two numbers from the list add up to k.

For example, given [10, 15, 3, 7] and k of 17,
return true since 10 + 7 is 17

If multiple combinations satisfied the condition then print
all combinations.

Input format:
First input is list seperated by one comma and space.
Ex: 1, 2, 3, 4, 5
Second input is the value of k

Output format:
if only one combination satisfied the condition
item1 item2

if more combinations satisfied the condition
item1 item2
item1 item2
item1 item2
